Hello and Welcome!

Kotlin/Native is all the hype these days. But as with every new technology just before its official release, things are moving fast. The documentation might be a bit out-of-date, some of it is missing, and perhaps the code examples aren't as clear.

This website tries to be your Kotlin/Native ELI5. It assumes you're familiar with Kotlin and Gradle. It takes you from a very simple Hello, World program to a slightly more involved Hello, libcurl program, to the pinnacle of Kotlin/Native - a multiplatform project that runs Kotlin code on an Android and iOS: Hello, Android & iOS.

This website, including its content, is available on GitHub. You're more than welcome to report (or even patch) typos, mistakes or misunderstandings.
